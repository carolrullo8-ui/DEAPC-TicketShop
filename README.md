# MarketingAuto — Sistema de Automatização de Marketing

## Descrição
O MarketingAuto é uma aplicação web que permite às empresas criar
e gerir campanhas de marketing digital de forma automatizada.
A aplicação permite gerir contactos (leads), criar e enviar campanhas,
e analisar o envolvimento dos destinatários com as campanhas enviadas.

## Objetivos
- Permitir a criação e gestão de campanhas de marketing
- Gerir a base de dados de leads (contactos potenciais)
- Rastrear o envolvimento dos leads com as campanhas
- Analisar os resultados das campanhas (aberturas, cliques, conversões)
- Segmentar leads por categorias para campanhas mais direcionadas

## Atores / Perfis de Utilizador

### 1. Gestor de Marketing
Utilizador principal da plataforma. Pode criar e gerir campanhas,
importar e gerir leads, consultar relatórios de envolvimento
e analisar o desempenho das campanhas.

### 2. Administrador
Responsável pela gestão da plataforma. Gere os utilizadores,
define permissões e acede a todos os relatórios e dados.
